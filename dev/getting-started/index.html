Getting started

Installation

Install CodeEdit.jl from the Julia package manager. If the package is not yet registered, add it from its repository URL:

pkg> add https://github.com/perrutquist/CodeEdit.jl

Loading the package

julia> using CodeEdit

Creating a handle

Use Handle to point at the block containing a source location:

julia> h = Handle("examples/MyPackage.jl", 10)ERROR: ArgumentError: line is outside file bounds: 10

If line 10 is inside a function, h points to the whole function block, not just that line.

Display the handle to inspect the code:

julia> hERROR: UndefVarError: `h` not defined in `Main`
Suggestion: add an appropriate import or assignment. This global was declared but not assigned.

or convert it to a string:

julia> source = string(h)ERROR: UndefVarError: `h` not defined in `Main`
Suggestion: add an appropriate import or assignment. This global was declared but not assigned.

Listing handles in files

List all blocks in one file:

julia> hs = handles("examples/MyPackage.jl")2 handles
# examples/delete.jl:
  1 - 3: function helper(x) return x + 1 end
  EOF:

List all blocks in files matching a glob:

julia> hs = handles("examples", "*.jl")8 handles
# examples/delete.jl:
  1 - 3: function helper(x) return x + 1 end
  EOF:

# examples/insert-after.jl:
  1 - 3: function foo(x) x + 1 end
  5 - 7: function helper(x) return x + 1 end
  EOF:

#invalid:
  #invalid
  #invalid
  #invalid

Follow Julia include statements recursively:

julia> hs = handles("examples/MyPackage.jl"; includes = true)2 handles
# examples/delete.jl:
  1 - 3: function helper(x) return x + 1 end
  EOF:

Getting the end-of-file handle

Use eof_handle when inserting new code at the end of a file:

julia> h = eof_handle("examples/append-example.jl")# examples/insert-after.jl EOF:
julia> edit = InsertBefore(h, raw"""
       function new_function()
           return nothing
       end
       
       """)Unsupported edit: ArgumentError: invalid handle

Searching text

Search within a set of handles:

julia> hs = handles("examples", "*.jl")8 handles
# examples/delete.jl:
  1 - 3: function helper(x) return x + 1 end
  EOF:

#invalid:
  #invalid
  #invalid
  #invalid
  #invalid
  #invalid
  #invalid
julia> matches = search(hs, "old_function_name")0 handles
julia> matches0 handles

The search result is a Set of handles, so you can inspect or edit each matching block.
